# TalentSync AI

## Smart Resume Screening & Job Match Engine

TalentSync AI is an AI-powered resume screening system that analyzes resumes against job descriptions and provides intelligent matching insights.

---

## Features

✅ Resume PDF Upload

✅ Resume Text Extraction

✅ Job Description Analysis

✅ TF-IDF Based Matching

✅ Cosine Similarity Scoring

✅ Skill Matching

✅ Missing Skill Detection

✅ Resume Improvement Recommendations

✅ Analytics Dashboard

---

## Tech Stack

- Python
- Flask
- Scikit-Learn
- PyPDF2
- HTML
- Bootstrap 5

---

## Project Workflow

1. Upload Resume
2. Enter Job Description
3. Extract Resume Content
4. Calculate Match Score
5. Detect Skills
6. Generate Recommendations
7. Display Dashboard Results
